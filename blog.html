Scope of `var`, `let`, and `const` 

                Var: var variables are function-scoped, meaning they are visible and accessible within the function in which they are declared,
                regardless of block scope.

                Var: let variables are block-scoped, which means they are only accessible within the block (curly braces) in which they are defined.

                Var: Variables declared with const are hoisted but, similar to let, are not initialized until the actual declaration statement is encountered.

Use cases of `null` and `undefined` 

                Null:null is a special value in JavaScript that represents the intentional absence of any object value or no value at all.

                undefined: Undefined is a primitive value in JavaScript that represents the absence of a value or the uninitialized state of a variable.

What is REST API? 

                REST API,is an architectural style for designing networked applications. It is an approach to building web services that are lightweight,
                scalable, and easy to maintain. REST is not a technology but a set of constraints and principles that define how web services should behave and
                interact with each other.
